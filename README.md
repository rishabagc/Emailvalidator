# Emailvalidator

#Python 3.6+ and Django versions 2.2+

## create virtualenv
`python3 -m virtualenv venv`

## activate virtualenv
`venv/bin/activate`

## Install packages from requirement.txt
`pip install -r requirement.txt`

### start project: 
`python manage.py runserver`
